To run app, 

1. Open terminal 
2. type: createdb inventory 
3. type: psql -d inventory -f db/inventory.sql 
4. type: python3 console.py 
5. type: quit: command + d
6. type: python3 -m flask run 
7. open the browser and type: 127.0.0.1:5000 in address bar
